## Algebraic Notation

El board se compone de celdas marcadas con las letras A - H y los numeros 1 - 8.
Donde la celda abajo a la izquierda es A1

Ejemplo:

|     | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 |
|-----|---| ---| ---| ---| ---| ---|---| ---|
| H   |
| G   |
| F   |
| E   |
| D   |
| C   |
| B   |
| A   |
